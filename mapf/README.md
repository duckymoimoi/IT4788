# Bài tập cộng điểm MAPF - League of Robot Runner
**Sinh viên:** [Dương Quốc Chính] - MSSV 202400033

### Yêu cầu đã thực hiện:
1. Chạy mã nguồn nhóm giải nhất Team_No_Man's_Sky trên map Warehouse.
2. Sửa logic: Robot dừng tại đích X trong (X % 100) giây.
3. Video minh chứng: trong repo này

### Cách chạy:
- Build bằng Code::Blocks + CMake + Boost 1.90.0.
- Lệnh chạy: `lifelong.exe --map maps/warehouse_large.map --task tasks/warehouse_large_10.task --output output.json`
- Hiển thị: Dùng PlanViz với file output.json.
