TONG HOP API HE THONG DIEU PHOI BENH VIEN - HOSPITAL NAVIGATION SYSTEM

QUY UOC:
  - METHOD  : POST / GET / DELETE / PATCH
  - AUTH    : X = Public (khong can token), O = Private (can Bearer JWT token)
  - MVP     : SU DUNG = giu lai cho giai doan chay thu
              BO QUA   = khong lam trong MVP, co the bo sung sau
              GIA LAP  = lam API nhung tra data tinh, khong ket noi he thong that
