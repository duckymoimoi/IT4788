================================================================================
QUY TRÌNH TRIỂN KHAI — 12 SLICE
Hospital Navigation & Coordination System
================================================================================

QUY TRÌNH CHUNG MỖI SLICE
================================================================================

  Bước 1: Schema    — Tạo struct Go trong schema/xxx.go
  Bước 2: Migrate   — Thêm vào database.go AutoMigrate()
  Bước 3: Seed      — Tạo dữ liệu mẫu (nếu cần)
  Bước 4: Repo      — Tạo repository/xxx_repo.go (truy vấn DB)
  Bước 5: Service   — Tạo service/xxx_service.go (logic nghiệp vụ)
  Bước 6: Handler   — Tạo handler/xxx_handler.go (nhận request, trả response)
  Bước 7: Routes    — Đăng ký endpoint trong handler/routes.go
  Bước 8: Test      — Viết test và chạy kiểm tra
  Bước 9:   Checkpoint — Build + test pass → tiến sang slice tiếp

Nếu slice không cần schema mới → bỏ qua bước 1-3.
Nếu repo/service đã có → chỉ bổ sung method mới.

CẦN TẠO MỚI:
  Schema (thêm vào schema/route.go):
    - routes (T18)
        Trường: route_id (PK auto), user_id (FK), mode_id,
                start_node_id, destinations_json (TEXT, mảng node codes),
                total_distance_m, estimated_time_s,
                status (active/completed/cancelled),
                created_at, updated_at
    - route_paths (T19)
        Trường: path_id (PK auto), route_id (FK), step_order,
                node_id (FK map_nodes), floor_id (FK),
                x (pixel), y (pixel), distance_m,
                instruction (text: Rẽ trái/Đi thẳng...)
  Migrate:
    - database/database.go: thêm &schema.Route{}, &schema.RoutePath{}
  Repository (bổ sung route_repo.go):
    - CreateRoute(), FindRouteByID()
    - CreateRoutePaths() (bulk insert)
    - FindRoutePaths(routeID)
  Service (bổ sung route_service.go):
    - PreviewPath(start, destinations, mode)
        → Chạy Dijkstra từng cặp, trả path_coords + distance + ETA
        → KHÔNG lưu DB
    - RouteOrdered(userID, start, destinations, mode)
        → Giống Preview nhưng INSERT routes + route_paths
    - GetSteps(routeID)
        → Đọc route_paths, trả danh sách instruction
    - GetETA(routeID, currentNodeID)
        → Tính distance còn lại ÷ (1.2 × speed_factor)
  Handler (bổ sung route_handler.go):
    - Preview()       → POST /api/route/preview
    - RouteOrdered()  → POST /api/route/ordered
    - GetSteps()      → GET  /api/route/get_steps?route_id=
    - GetETA()        → POST /api/route/get_eta
  Routes:
    - handler/routes.go: thêm 4 endpoint (cần JWT)
  Test:
    - Test preview với map data thật
    - Test ordered → kiểm tra DB có route + paths
    - Test get_steps → kiểm tra instruction
    - Test get_eta → kiểm tra thời gian hợp lý
